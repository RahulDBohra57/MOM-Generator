# AI-Powered Modular Minutes of Meeting (MoM) Generator

**AI-Powered Modular MoM Generator** is an intelligent document automation system that converts unstructured meeting notes ‚Äî including handwritten images, scanned PDFs, and text files ‚Äî into standardized, structured, and downloadable Minutes of Meeting (MoM) within minutes.

---

## Executive Summary

In industries such as construction, infrastructure, civil engineering, and project management, meeting documentation remains heavily manual and inconsistent.

Teams often rely on:
- Handwritten notebooks  
- WhatsApp images  
- Scanned PDFs  
- Informal text notes  

These fragmented formats make it difficult to:
- Track accountability  
- Assign ownership  
- Monitor deadlines  
- Maintain auditable documentation  

The **AI-Powered Modular MoM Generator** solves this problem by using **Vision AI + Generative AI** to automatically extract, structure, and standardize meeting minutes across multiple input formats.

## Business Objective

The key objectives of this solution are to:

1. Automate extraction of handwritten and digital meeting notes  
2. Generate **standardized, tabular MoMs** with defined fields  
3. Support **multi-format inputs** (images, PDFs, Word, text)  
4. Enable **instant Word and PDF downloads**  
5. Create an auditable digital record of discussions and decisions  
6. Reduce manual effort and improve execution clarity

## Solution Overview

An end-to-end AI-powered MoM automation platform built using:

- **Google Gemini Vision API** for OCR and image understanding  
- **Prompt-engineered LLM workflows** for structured MoM generation  
- **Streamlit** for a no-code, user-friendly interface
